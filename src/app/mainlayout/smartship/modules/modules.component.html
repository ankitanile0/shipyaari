Shpoify

Marketplace

Use the following instructions to integrate Shopify:

  • navigate_next First you need to Log in to Shopify Admin Panel.
  • navigate_next Go to Apps.
  • navigate_next By Scrolling down you will find a Manage private apps link.
  • navigate_next From here, You need to click on the Create new private app button.
  • navigate_next Give some name in your Private app for recognise. (Like: Shipyaari).
  • navigate_next Next, you need to give admin access for some specific APIs:
  • navigate_next Read Access: Fulfillment Services & Inventory.
  • navigate_next Read & Write Access: Products, Product Listings, Assigned Fulfillment Orders, Customers, Draft Orders, Orders, Merchant Managed Fulfillment Orders, Order Editing, Store Content, Third-Party Fulfillment Orders.
  • navigate_next Save your changes to get API credentials (API Key, Password, and Shared Secret).
  • navigate_next Note: For Store URL, use: https://mystorename.myshopify.com
  • navigate_next Paste the same credentials here to integrate the shopify.

Magento V1 Module

Marketplace

Use the following instructions to integrate Magento:

  • navigate_next First you need to Login into Magento Admin Panel
  • navigate_next Then go to the System -> Web Services -> SOAP/XML -RPC Users.
  • navigate_next Then click on "Add New User".
  • navigate_next Then under User Info tab, fill all required details like user name, first name etc.
  • navigate_next Then click on the "Save User" shown at the top right hand side.
  • navigate_nextAgain go to System -> Web Services -> SOAP/XML -RPC Roles
  • navigate_next Click on Add new roles Button.
  • navigate_next Then click on Role Info tab and type role name like adminship
  • navigate_next Click on Role Resources tab, And select Resource Access "All" from drop down menu and click "Save Role".
  • navigate_next Now again Go to System -> Web Services -> SOAP/XML-RPC Users.
  • navigate_next Then click on user name which has been entered.
  • navigate_next Under User Role tab, click on assigned role name like adminship, like you have entered before.

Now inside Shipyaari Dashboard:

  • navigate_next Fill in the Parameters as saved from the Magento Panel Store URL : The URL of your store. API Key & API Username : API details of your Magento Panel

Woo-Commerce

Marketplace

Use the following instructions to integrate Woo-Commerce:

  • navigate_next Enter your store name/website name.
  • navigate_next Now enter your store URL.
  • navigate_next Enable REST API in your WooCommerce plugin.To enable REST API in WooCommerce, Navigate to Woocommerce>Settings>Advanced>Legacy API.
  • navigate_next Now click on Enable Rest API and save the settings.
  • navigate_next Now click on add key button and save the consumer secret and consumer key.
  • navigate_next Paste those details here in shipyaari that you have copied from wooCommerce admin panel.

Opencart

Marketplace

Use the following instructions to integrate opencart:

In Opencart Admin:

  • navigate_next Go to your opencart admin panel
  • navigate_next Enable API in your Opencart admin panel.
  • navigate_next To enable API in Opencart, Navigate to System>Users>API.
  • navigate_next Add API username and API key and status Enable.

In shipyaari Dashboard:

  • navigate_next Add those details that where copy from your opencart admin panel.
  • navigate_next Download opencart plugin from given link: Download Plugin
  • navigate_next Please unzip and upload the shipyaari plugin folder inside catalog > controller within your file manager. for example http://yourdomain.com/catalog/controller
  • navigate_nextAll done, now scyn order and confirm.

Wix

Marketplace

Use the following instructions to integrate Wix:

  • navigate_nextClick on Intregrate now or Update now button It will redirect you to wix website.
  • navigate_nextClick on add to site button
  • navigate_next If you are not logged in wix will ask you to login first.
  • navigate_nextAfter login to your account wix will ask you to select store which you want to intregrate with shipyaari.
  • navigate_next Select the store then you will redirect to shipyaari website for identification (Note: You must login to your shipyaari account in a same browser).
  • navigate_next Then you will be taken to wix website to grant some permissons.
  • navigate_next Click on add to site button then you will be redirect to shipyaari website.
  • navigate_next All done, now scyn order and confirm.
